🎓 Academic Management Platform (Student Management System)

A scalable full-stack web application designed to manage academic workflows including student enrollment, course scheduling, fee tracking, and role-based access control.

⚙️ Built With
⚛️ React.js
🟢 Node.js
🚀 Express.js
🔐 JWT Authentication
🗄️ MySQL Database
🔑 RBAC (Role-Based Access Control)
📡 RESTful API Architecture

✨ Features
• Multi-role authentication system (Admin / Teacher / Student / Accountant)
• Course enrollment with transactional integrity
• Academic workflow lifecycle management
• Fee & invoice tracking module
• Timetable conflict detection
• Notification system
• Audit logging


---

## 🔐 User Roles

- Super Admin  
- Admin  
- Teacher  
- Student  
- Accountant  

📦 Structured using a modular MVC-inspired backend architecture for scalability and maintainability.
